# 开发路线图

## 当前状态

- ✅ 里程碑一：工程骨架
- ✅ 里程碑二：最小训练闭环
- ⬜ 里程碑三：最小推理闭环
- ⬜ 里程碑四：特征扩展

---

## 里程碑一：工程骨架 ✅

建立 `src/lottery/` 包结构，明确模块边界与工程约定。

- 建立 `cli / config / domain / data / models / training / inference` 包
- 实现 TOML 配置加载器（含默认值回退）
- 实现 argparse CLI 入口（`train`、`predict` 子命令）
- 统一输出目录规范（`output/models/`、`output/logs/`、`output/summaries/`）

---

## 里程碑二：最小训练闭环 ✅

跑通数据读取 → 样本构造 → 模型训练 → 产物输出的完整链路。

- 从 `data/raw_ssq.txt` 读取双色球历史数据（仅使用前 9 列）
- 定义 `LotteryRecord` 领域对象
- 实现 `LotteryDataset`（PyTorch Dataset，`seq_len=10`）
- 实现 `LotteryLSTM` 模型（7 维输入，64 维隐藏层，2 层 LSTM）
- 实现 `Trainer`（MSE Loss + Adam 优化器）
- 集成 TensorBoard（记录 batch/epoch 损失）
- 实现 `save_model`（保存 `model.pt` + `metadata.json`）

---
